In conducting framing analysis of an issue, we suggest users first come up with a list of “frames” that guide the discussion of the issue both inductively and deductively. Step 1 deals with the inductive understanding of the data, that is, the user should first get an idea about the text corpus regarding the issue. If the project involves a large amount of data, the LDA (Latent Dirichlet Allocation) topic modeling is a recommended method for you to explore main “topics” in the data. The LDA topic modeling is an unsupervised learning method, which means the user does not need to provide labeled data to build the computational models. The unit of analysis in the LDA topic modeling is called a “document.” It can be a news article, an email, or a tweet. Note that a LDA topic is different from a frame, which is more abstract and many are formally defined in the communication literature. The goal is to draw upon the “topic” information from the LDA topic modeling results (Step 1; inductive) and the review of relevant literature (Step 2; deductive) to decide frames to be analyzed (Step 3-5).
